# gh-apps-token

The CLI tool wraps bradleyfalzon/ghinstallation/v2.

## Usage

```
$ export GITHUB_APPS_PRIVATE_KEY=$(cat /path/to/your/private_key_file)
$ gh-apps-token --app-id 100 --installation-id 2000
ghs_xxxxx
```
